# FreePlay

FreePlay showcases free online games using the FreeToGame API.

## Features

- Home page with a featured game and popular game cards
- Games page with search, filter, and pagination support
- Game details page with additional information and a play link
- Favourites page for saved games using local storage
- Contact page with game suggestion form and saved suggestions
- Search and filter games on the `games.html` page.
- Save your favorite games and view them on the `favourites.html` page.
- Game data is loaded from the FreeToGame API.
- Saved favourites and suggestions are stored locally in the browser.
